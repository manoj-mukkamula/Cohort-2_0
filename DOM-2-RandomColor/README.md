# Random Color

DOM 2 : RandomColor — Generate and apply random colors to the box.

A small, interactive DOM project that generates a random color and applies it to an element (box). The current color code is displayed so you can see the RGB value.

## Demo
Open the live demo in your browser:
https://manoj-mukkamula.github.io/Cohort-2_0/DOM-2-RandomColor/

## Features
- Generate a random color
- Apply the generated color to the UI (target element)
- Simple, accessible controls and visual feedback

## Files
- `index.html` — markup and project entry.
- `style.css` — styling for the Random Color UI.
- `script.js` — JavaScript logic that generates colors and updates the UI.

## How to run locally
1. Clone the repository:
   git clone https://github.com/manoj-mukkamula/Cohort-2_0.git
2. Open the folder `DOM-2-RandomColor` and open `index.html` in your browser:
   - Double-click `DOM-2-RandomColor/index.html` to open it directly, or
   - Serve the folder with any static server or editor extension (for example: Live Server in VS Code) and visit the corresponding local URL.

## Notes
This project demonstrates core DOM and JavaScript fundamentals:
- Selecting elements
- Updating HTML content
- Changing CSS styles programmatically
- Handling user events
- Generating random values
