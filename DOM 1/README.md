# Counter

DOM 1 : Counter - Increment, Reset, Decrement of Numbers.

A small, interactive DOM project that implements a simple counter with Increase, Decrease and Reset controls. The displayed number changes color to indicate positive (green), negative (red), or zero (gray).

## Demo
Open the live demo in your browser:
https://manoj-mukkamula.github.io/Cohort-2_0/DOM%201/

## Features
- Increase counter
- Decrease counter
- Reset counter to zero
- Color-coded value:
  - Positive → green
  - Negative → red
  - Zero → gray

## Files
- `index.html` — markup and project entry.
- `style.css` — styling for the counter UI.
- `script.js` — JavaScript logic that updates the counter and colors.

## How to run locally
1. Clone the repository:
   git clone https://github.com/manoj-mukkamula/Cohort-2_0.git
2. Open the folder `DOM 1` and open `index.html` in your browser:
   - Double-click `DOM 1/index.html` to open it directly, or
   - Serve the folder with any static server or editor extension (for example: Live Server in VS Code) and visit the corresponding local URL.
