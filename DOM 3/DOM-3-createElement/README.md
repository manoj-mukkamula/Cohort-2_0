# DOM 3: Create Elements ‚Äì Randomly Generated Colorful Boxes

A simple **Document Object Model (DOM)** project demonstrating how to dynamically create and style elements using **JavaScript**. Each click on the button generates a new square with a random background color, a random position, and a random rotation on the screen.

## ‚ú® Features

* Creates new elements dynamically using `document.createElement()`.
* Assigns **random background colors** to each box.
* Applies **random rotation** (0‚Äì360 degrees) for visual variation.
* Sets a **random position** to place the box anywhere on the screen.
* Provides a clean and minimalist **UI** centered around a trigger button.

## üìö Technical Notes

This project reinforced the following **core JavaScript and DOM concepts**:

* **DOM Manipulation:** Selecting existing elements and creating new elements dynamically.
* **Styling with JavaScript:** Modifying the `style` properties of elements (e.g., `backgroundColor`, `transform`, `top`, `left`).
* **Event Handling:** Implementing an `event listener` (`click`) on the button element.
* **Random Value Generation:** Utilizing `Math.random()` to generate unpredictable values for color (RGB), position, and rotation.
